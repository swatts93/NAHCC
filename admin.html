⚙ Admin Setup Helper
Complete these steps once before deploying your admin panel.
  1. Type your chosen admin password below and click Generate Hash. Copy the hash that appears.
  2. Open admin.html in a text editor (or click the pencil icon on GitHub). Find the PASSWORD_HASH line near the top of the <script> section and replace REPLACE_WITH_HASH with the hash you copied.
  3. While you have the file open, also set GITHUB_OWNER to your GitHub username and confirm GITHUB_REPO matches your repo name.
  4. Save and commit admin.html. Then refresh this page — the login screen will appear.

🔑 Connect to GitHub
This is a one-time setup on this browser. Your token is saved locally and never stored in the code or repository.
  1. Go to github.com/settings/personal-access-tokens/new (opens in a new tab).
  2. Give it a name like Partner Directory Admin. Set Expiration to your preference (1 year is reasonable).
  3. Under Repository access, choose Only select repositories and pick your partner-directory repo.
  4. Under Repository permissions, find Contents and set it to Read and write. Leave everything else as No access.
  5. Click Generate token, copy it, and paste it below.
Paste the token exactly as GitHub shows it. You will not be able to see it again on GitHub after leaving that page.
